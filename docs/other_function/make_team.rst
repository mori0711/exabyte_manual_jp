==============
チームの作成方法
==============

組織アカントの下に、チームを作成しグループ管理することができます。

| ここでは、チームの作成方法を説明します。
| チームを作成する場合、TEAMSの項目を選択します。

  +--------------------------------------------------------------------------+
  | .. image:: ./imgs/make_team_000.png                                      |
  |    :scale: 50 %                                                          |
  |    :align: center                                                        |
  +--------------------------------------------------------------------------+
  
| Createボタンを押します。

  +--------------------------------------------------------------------------+
  | .. image:: ./imgs/make_team_001.png                                      |
  |    :scale: 50 %                                                          |
  |    :align: center                                                        |
  +--------------------------------------------------------------------------+

| チーム名や権限の設定を行います。

  +--------------------------------------------------------------------------+
  | .. image:: ./imgs/make_team_002.png                                      |
  |    :scale: 50 %                                                          |
  |    :align: center                                                        |
  +--------------------------------------------------------------------------+

.. csv-table::
   :header-rows: 1
   :widths: 5, 5
   :align: center

   項目名, 詳細
   read, 読み込み
   edit, 書き込み
   comment, コメント追加
   execute, 計算実行

| 全ての権限を与える場合には、全て選択します。

  +--------------------------------------------------------------------------+
  | .. image:: ./imgs/make_team_003.png                                      |
  |    :scale: 50 %                                                          |
  |    :align: center                                                        |
